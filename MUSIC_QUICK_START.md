# 🎵 音樂庫快速啟動指南

## 🚀 5 分鐘快速上手

### 步驟 1: 準備音樂文件 (1 分鐘)

確認你的音樂文件已放置在 `public/music` 目錄：

```
public/music/
├── 鋒兄的傳奇人生/
├── 鋒兄進化Show🔥/
├── 塗哥水電王子爆紅/
└── 最瞎結婚理由/
```

✅ **已完成**: 48 首音樂已準備就緒

### 步驟 2: 設置環境變數 (1 分鐘)

在 `.env` 文件中添加：

```bash
NETLIFY_SITE_ID=你的站點ID
NETLIFY_AUTH_TOKEN=你的認證令牌
```

💡 **提示**: 從 Netlify 控制台獲取這些值

### 步驟 3: 上傳音樂 (2 分鐘)

**Windows 用戶:**
```bash
upload-music.bat
```

**其他用戶:**
```bash
npm run upload-music
```

⏳ **等待**: 上傳 48 首音樂大約需要 2-3 分鐘

### 步驟 4: 啟動應用 (30 秒)

```bash
npm run dev
```

### 步驟 5: 訪問音樂庫 (30 秒)

1. 打開瀏覽器訪問 `http://localhost:3000`
2. 點擊側邊欄的「🎵 音樂庫」
3. 開始享受音樂！

## 💡 小技巧

### 預載所有音樂
在音樂庫頁面點擊「預載所有音樂」按鈕，將所有音樂快取到本地，享受離線播放。

### 清除快取
如果遇到播放問題，可以使用「清除所有快取」功能重新載入音樂。

### 移動設備
音樂庫完全支援移動設備，可以在手機或平板上流暢使用。
